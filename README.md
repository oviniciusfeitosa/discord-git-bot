# Discord Git Bot

Discord Git Bot using Webhooks.

## Prerequisites

- CURL
- Discord Server Admin privileges

## Configure

- Install CURL

```sh
sudo apt install curl
```

- Create a discord bot inside Discord Server

- Set your configs inside `config.sh`.

## Send message - Curl

```sh
source start.sh
```
